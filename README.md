# Bot Maker – Full Stack Project

This project is a **Full Stack Bot Management Application** built using **Spring Boot (Backend)** and **React (Frontend)**.

The application allows users to **register, login, and manage bots** using secure APIs.

---

## Tech Stack

### Backend

* Java
* Spring Boot
* Spring Security
* JWT Authentication
* Maven
* H2 / MySQL Database

### Frontend

* React
* Axios
* HTML
* CSS
* JavaScript

---

# Project Structure

bot-maker
│
├── src → Spring Boot Backend
├── botmaker_frontend → React Frontend
├── pom.xml
└── README.md

---

# Backend Setup (Spring Boot)

### 1. Navigate to project folder

cd bot-maker

### 2. Run the application

mvn spring-boot:run

Backend will start on:

http://localhost:8080

---

# Frontend Setup (React)

### 1. Navigate to frontend folder

cd botmaker_frontend

### 2. Install dependencies

npm install

### 3. Start React app

npm start

Frontend will start on:

http://localhost:3000

---

# API Documentation

API endpoints can be tested using **Swagger UI**.

Swagger URL:

http://localhost:8080/swagger-ui/index.html

---

# Features

User Registration
User Login with JWT Authentication
Secure API Endpoints
Bot Management APIs
Frontend UI for interacting with APIs

---

# Security

* JWT based authentication implemented
* Password encryption using BCrypt
* Protected endpoints using Spring Security
